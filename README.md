## 概述

B2B 销售智能助理的完整版本，覆盖从线索挖掘到赢单复盘的完整销售流程，包含 22+ 项核心能力。在分析竞争版基础上新增赢单/输单复盘与交接、客户动态舆情监控、标杆案例智能匹配、行业政策红利提醒、案例库维护和销售团队负责人专属功能。

## 安装

将 `Sloth-Sales-Eido/` 整个目录复制到 `~/.qoderwork/skills/` 下即可。

```bash
cp -r Sloth-Sales-Eido ~/.qoderwork/skills/
```

从任意旧版本升级：直接覆盖文件，运行 `init_db.py` 自动补建缺失的表，已有数据完整保留。

首次使用时，Skill 会自动引导初始化。也可手动运行：

```bash
python ~/.qoderwork/skills/Sloth-Sales-Eido/scripts/init_db.py
```

## 功能详细说明

### 一、客户挖掘与信息补全（v1.0）

按区域、行业、规模等条件搜索潜在客户，联网搜索后整理成结构化信息，支持批量加入漏斗。可指定的筛选条件包括城市/区域、行业、规模（人员数/营收）和资质标签（专精特新、规上企业等）。

### 二、客户信息管理（v1.0）

增删改查客户记录，管理决策链（角色、影响力、态度），维护联系人信息，设置重点关注标记。当客户合同到期日不足6个月时自动提示切入窗口期。

### 三、销售漏斗推进与活动记录（v1.0）

管理销售漏斗阶段（线索→初步接触→需求确认→方案演示→商务谈判→赢单/输单），支持一句话快速记录跟进活动，助理自动解析活动类型、内容摘要、结果和建议操作。

漏斗阶段与默认概率：

| 阶段 | 默认概率 | 说明 |
|------|----------|------|
| 线索 | 10% | 初步识别的潜在客户 |
| 初步接触 | 20% | 已进行首次沟通 |
| 需求确认 | 40% | 明确了客户需求 |
| 方案演示 | 60% | 已展示产品方案 |
| 商务谈判 | 80% | 进入价格/条款协商 |
| 赢单 | 100% | 成功签约 |
| 输单 | 0% | 未能成交 |

### 四、销售方法论应用（v1.0）

支持三种主流 B2B 销售方法论，可随时切换：

| 方法论 | 核心框架 | 适用场景 |
|--------|----------|----------|
| SPIN | 情境→难点→暗示→效益 | 挖掘深层需求 |
| BANT | 预算→权限→需求→时间 | 快速评估商机质量 |
| MEDDIC | 指标→买家→标准→流程→痛点→支持者 | 复杂大单分析 |

可自动基于所选方法论为特定客户生成结构化提问清单。

### 五、数据导出与备份（v1.0）

一键导出 customers、sales_funnel、activity_log 三张表为 Excel 文件（每表一个 Sheet）。每周五系统自动提醒备份。

### 六、智能拜访准备（v2.0）

触发词"准备拜访【客户名】"后，自动生成涵盖以下 5 个板块的一页纸简报：

1. **公司概况与动态**：基础信息 + 联网搜索近期新闻
2. **当前方案与痛点**：基于 current_solution 字段推测业务痛点
3. **决策链切入点**：关键决策人态度分析与接触策略建议
4. **结构化提问清单**：基于所选方法论自动生成针对性提问
5. **参考案例**：匹配同行业同规模的成功案例

支持进阶用法：基于拜访准备内容生成会议邀约邮件草稿。

### 七、个性化价值主张生成器（v2.0）

触发词"生成价值主张【客户名】"后，基于客户的当前方案痛点输出 3 条可直接使用的定制价值点。适用于邮件标题、方案摘要页、电话开场白和演示开场。

### 八、会议纪要智能处理（v2.0）

提供会议内容后，自动提取竞品、预算、时间节点等关键信息，更新数据库字段（current_solution、estimated_deal_value、decision_chain、key_extracted_info 等），并生成结构化纪要（讨论要点 + 结论 + 待办事项），自动存入 activity_log。检测到积极信号时会给出鼓励和里程碑建议。

### 九、关键日期与合规提醒（v2.0）

从跟进记录中自动识别关键日期（报价有效期、合同到期日、投标截止日、演示安排、付款节点），存储到 key_date 字段。每日首次对话时展示未来 7 天的关键日期清单。

### 十、商机健康度评分与提醒（v3.0）

基于跟进间隔、行动计划、关系趋势、痛点匹配和预算信号五个维度自动计算 0-100 分（基础分70分），每日首次对话提醒低于 50 分的客户。

评分维度：

| 维度 | 规则 | 分数影响 |
|------|------|----------|
| 跟进间隔 | 超过7天 / 超过14天 | -15 / -30 |
| 行动计划 | 无 next_action | -20 |
| 关系趋势 | relationship_score 下降 | -10 |
| 痛点匹配 | 客户有老旧系统或明确需求 | +15 |
| 预算信号 | 有预算信号 | +20 |

当 next_contact_date 过期>7天且健康分<40时，标注为"待复活客户"。

### 十一、商机"阻力点"智能诊断（v3.0）

综合活动日志、决策链反对者、竞品介入、预算缺失和方法论缺口五个维度，输出结构化的卡点报告和突破建议。触发词"分析一下【客户名】为什么卡住"。

### 十二、竞品应对与话术生成（v3.0）

按"认同→差异化→风险→案例→邀约"五步结构，结合客户实际情况生成话术。触发词"客户说【竞品】更好，怎么回应？"。

### 十三、协作快照生成（v3.0）

200字内的结构化客户摘要，涵盖基本信息、当前方案、核心痛点、关键人和当前阶段，可直接复制转发给售前顾问、技术专家或上级领导。

### 十四、内部求助话术生成（v3.0）

一键生成可直接复制到企业 IM 的 @同事求助话术，包含客户背景、痛点、演示对象、时间安排和重点展示内容。

### 十五、决策人角色化沟通（v3.0）

根据收件人角色自动调整内容重点和语言风格：

| 角色 | 关注重点 | 语言风格 |
|------|----------|----------|
| 经济决策者 | ROI、战略价值、风险控制 | 简洁数据驱动 |
| 技术评估者 | 性能、SLA、安全、集成 | 技术深度 |
| 使用者 | 易用性、培训、效率 | 场景化 |
| 采购执行者 | 价格、条款、资质 | 商务规范 |

### 十六、个人业绩仪表盘与行为反馈（v3.0）

一句"我的业绩"即可看到新增线索、赢单金额、漏斗总额等核心指标及环比变化，并附带行为诊断建议。

### 十七、赢单/输单复盘与交接（v4.0 新增）

**赢单**：自动引导生成客户交接清单（签约承诺、联系人沟通偏好、风险点、服务建议），支持设置履约节点日期（首付款、启动会、验收等），这些日期会纳入每日提醒。

**输单**：自动追问两个问题（最终选了哪家、主要原因），以 JSON 格式存入日志，积累后可通过"输单原因分析"查看趋势分布。

### 十八、客户动态舆情监控（v4.0 新增）

对设置了"监控"标记的客户进行联网新闻扫描，识别高管变更、融资/上市、新工厂/扩产和负面信息，输出一句话简报 + 行动建议。触发词"查看我关注的客户动态"或"舆情扫描"。

v4.1 通过 Agent-Reach 全渠道搜索大幅增强舆情覆盖面，支持的渠道包括：

- **Exa AI**：企业新闻、行业动态深度搜索
- **微信公众号**：企业官方公众号文章、行业媒体报道
- **Twitter/微博**：品牌舆情、高管社交媒体动态
- **LinkedIn**：关键人职位变动、企业人事公告
- **Jina Reader**：行业媒体文章全文阅读与解析

### 十九、标杆案例智能匹配（v4.0 新增）

基于客户画像（行业、规模、采购品类、原有方案）匹配案例库中最相似的 1-2 个案例，输出客户化场景描述和建议讲述方式。案例库为空时引导建设。

### 二十、行业政策/红利提醒（v4.0 新增）

在拜访准备或查看客户时，自动联网检索当地产业扶持政策（如"智改数转"补贴等），作为与客户沟通的额外价值点。支持手动检索特定行业政策。

### 二十一、案例库维护（v4.0 新增）

支持逐条录入（助理引导填写行业、规模、原有方案、核心痛点、我方方案、量化收益、客户评价）和 CSV 批量导入两种方式建设案例库。

### 二十二、团队负责人专属功能（v4.0 新增）

仅在角色为"销售团队负责人"时激活。数据汇聚方式：各销售定期导出个人数据为 Excel，负责人上传后自动合并、去重、生成团队看板。

**18 项团队管理能力**：

看板与统计：团队漏斗看板、团队业绩仪表盘、活动量审计、销售预测分级（Commit/Best Case/Pipeline）。

预警与检测：撞单检测（模糊匹配重复跟进客户）、停滞商机预警（过期>14天按销售分组）、续约/增购预警、销售负荷监测（并行商机>15且周活动<5）。

分析与诊断：输单原因分析（统计 lost_to 和 reason 分布）、流程瓶颈诊断（各阶段转化率）、销售能力对标（五维度评分）、售前资源ROI排序（按"金额×概率"排序）。

情报与协作：外部情报扫描（Agent-Reach全渠道搜索行业招标/政策匹配客户库）、自定义预警规则、生成销售反馈（结构化认可与建议）、生成团队周报（漏斗+动态+风险+下周重点）、导出团队汇总表。
