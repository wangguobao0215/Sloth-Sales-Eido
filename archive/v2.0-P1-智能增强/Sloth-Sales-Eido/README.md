# B2B 销售智能助理 v2.1（智能增强版）

## 概述

在核心 CRM 基础上，增加智能拜访准备简报、会议纪要自动解析、个性化价值主张生成和关键日期自动提醒四大能力，让销售在见客户前后的效率大幅提升。

## 功能清单

### 核心 CRM 功能（继承自 v1.0，5项）

1. **客户挖掘与信息补全** — 按区域/行业/规模搜索并补全客户信息，支持 Agent-Reach 增强多平台联网挖掘
2. **客户信息管理** — 增删改查客户、决策链、联系人
3. **销售漏斗推进与活动记录** — 阶段管理、一句话快速记录
4. **销售方法论应用** — 支持 SPIN/BANT/MEDDIC 三种方法论
5. **数据导出与备份** — 一键导出 Excel

### v2.0 新增功能（4项）

6. **智能拜访准备（Pre-Call Brief）** — 一页纸客户简报，集成 Agent-Reach 实时新闻采集
7. **个性化价值主张生成器** — 基于客户痛点定制 3 条价值点
8. **会议纪要智能处理** — 自动提取关键信息并更新数据库
9. **关键日期与合规提醒** — 自动识别并追踪重要日期

## 安装

将 `Sloth-Sales-Eido/` 整个目录复制到 `~/.qoderwork/skills/` 下即可。

```bash
cp -r Sloth-Sales-Eido ~/.qoderwork/skills/
```

如果已安装 v1.0，直接覆盖即可，数据库兼容无需迁移。

首次使用时，Skill 会自动引导初始化。也可手动运行：

```bash
python ~/.qoderwork/skills/Sloth-Sales-Eido/scripts/init_db.py
```

## 功能详细说明

### 一、客户挖掘与信息补全（v1.0）

按区域、行业、规模等条件搜索潜在客户，联网搜索后整理成结构化信息，支持批量加入漏斗。可指定的筛选条件包括城市/区域、行业、规模（人员数/营收）和资质标签（专精特新、规上企业等）。

### 二、客户信息管理（v1.0）

增删改查客户记录，管理决策链（角色、影响力、态度），维护联系人信息，设置重点关注标记。当客户合同到期日不足6个月时自动提示切入窗口期。

### 三、销售漏斗推进与活动记录（v1.0）

管理销售漏斗阶段（线索→初步接触→需求确认→方案演示→商务谈判→赢单/输单），支持一句话快速记录跟进活动，助理自动解析活动类型、内容摘要、结果和建议操作。

漏斗阶段与默认概率：

| 阶段 | 默认概率 | 说明 |
|------|----------|------|
| 线索 | 10% | 初步识别的潜在客户 |
| 初步接触 | 20% | 已进行首次沟通 |
| 需求确认 | 40% | 明确了客户需求 |
| 方案演示 | 60% | 已展示产品方案 |
| 商务谈判 | 80% | 进入价格/条款协商 |
| 赢单 | 100% | 成功签约 |
| 输单 | 0% | 未能成交 |

### 四、销售方法论应用（v1.0）

支持三种主流 B2B 销售方法论，可随时切换：

| 方法论 | 核心框架 | 适用场景 |
|--------|----------|----------|
| SPIN | 情境→难点→暗示→效益 | 挖掘深层需求 |
| BANT | 预算→权限→需求→时间 | 快速评估商机质量 |
| MEDDIC | 指标→买家→标准→流程→痛点→支持者 | 复杂大单分析 |

可自动基于所选方法论为特定客户生成结构化提问清单。

### 五、数据导出与备份（v1.0）

一键导出 customers、sales_funnel、activity_log 三张表为 Excel 文件（每表一个 Sheet）。每周五系统自动提醒备份。

### 六、智能拜访准备（v2.0 新增）

触发词"准备拜访【客户名】"后，自动生成涵盖以下 5 个板块的一页纸简报：

1. **公司概况与动态**：基础信息 + 联网搜索近期新闻
2. **当前方案与痛点**：基于 current_solution 字段推测业务痛点
3. **决策链切入点**：关键决策人态度分析与接触策略建议
4. **结构化提问清单**：基于所选方法论自动生成针对性提问
5. **参考案例**：匹配同行业同规模的成功案例

> 当安装了 Agent-Reach Skill 时，"公司概况与动态"板块将通过 Agent-Reach 驱动的实时新闻采集（如 Exa AI 搜索）获取最新的企业新闻、融资动态和行业政策变化，使简报内容更加及时准确。

### 七、个性化价值主张生成器（v2.0 新增）

触发词"生成价值主张【客户名】"后，基于客户的当前方案痛点输出 3 条可直接使用的定制价值点。适用于邮件标题、方案摘要页、电话开场白和演示开场。

### 八、会议纪要智能处理（v2.0 新增）

提供会议内容后，自动提取竞品、预算、时间节点等关键信息，更新数据库字段（current_solution、estimated_deal_value、decision_chain、key_extracted_info 等），并生成结构化纪要（讨论要点 + 结论 + 待办事项），自动存入 activity_log。检测到积极信号时会给出鼓励和里程碑建议。

### 九、关键日期与合规提醒（v2.0 新增）

从跟进记录中自动识别关键日期（报价有效期、合同到期日、投标截止日、演示安排、付款节点），存储到 key_date 字段。每日首次对话时展示未来 7 天的关键日期清单。

## 适用场景

- 已搭建好数据底盘，需要提升拜访前后效率的销售团队
- 频繁进行客户拜访，需要快速生成准备材料
- 需要从会议中自动提取和归档关键商务信息
- 管理多个商机，需要自动追踪关键日期
