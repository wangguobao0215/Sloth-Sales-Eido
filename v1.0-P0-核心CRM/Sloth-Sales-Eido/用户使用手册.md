# B2B 销售智能助理 v1.1 用户使用手册

## 快速开始

### 第一步：首次启动

打开对话后，助理会自动引导您完成初始化设置：

1. 选择您的角色（一线销售 / 销售团队负责人）
2. 选择您销售的产品类型（软件、工业设备、专业服务等）
3. 选择偏好的销售方法论（SPIN / BANT / MEDDIC）
4. 选择是否导入现有客户名单

完成后，您的销售工作台即刻就绪。

### 第二步：添加第一个客户

您可以用以下任意方式开始：

**方式一：直接添加**
> "添加客户：苏州云创科技，电子元器件制造业，300人规模"

**方式二：批量挖掘**
> "帮我找苏州的制造业企业，营收5000万以上"

**方式三：导入名单**
> 上传一份 Excel/CSV 文件，助理会自动解析并补全信息

## 核心功能详解

### 一、客户挖掘与信息补全

**触发方式**：描述您的目标客户条件

**示例对话**：
> 用户："帮我找杭州地区营收5000万以上的制造业企业"
>
> 助理：（联网搜索后）已整理出34家潜在客户，预览如下...您可以下载完整 Excel，或说"全部加入漏斗"。

**可指定的筛选条件**：

- 城市/区域
- 行业
- 规模（人员数、营收）
- 资质标签（专精特新、规上企业等）

#### Agent-Reach 增强搜索

当安装了 Agent-Reach Skill 后，客户挖掘功能将自动升级为多平台深度搜索模式：

- **Exa AI 精准搜索**：通过 Exa AI 进行精准的企业信息搜索，获取更全面的公司背景数据
- **Jina Reader 年报解读**：通过 Jina Reader 获取并解读企业年报、财报等公开文档，快速提取关键经营指标
- **GitHub 技术栈调研**：通过 GitHub 搜索目标公司的开源项目和技术栈，为技术型销售提供切入点

**示例对话**：
> 用户："帮我深度调研一下华强实业的年报"
>
> 助理：（通过 Agent-Reach 的 Jina Reader 获取年报全文）已获取华强实业2024年年报关键信息...

### 二、客户信息管理

**查询客户**：
> "查看苏州云创科技的信息"
> "搜索姓名里有'云创'的客户"

**编辑信息**：
> "更新云创科技的联系人：IT总监李总，电话138xxxx"
> "设置云创科技的决策链：李总是技术评估者，影响力4，态度支持"

**监控客户**：
> "监控这家公司"（设置重点关注标记）

**合同到期提醒**：当客户的合同到期日不足6个月时，系统会自动提示"合同即将到期，可能是切入窗口期"。

### 三、销售漏斗推进与活动记录

**漏斗阶段说明**：

| 阶段 | 默认概率 | 说明 |
|------|----------|------|
| 线索 | 10% | 初步识别的潜在客户 |
| 初步接触 | 20% | 已进行首次沟通 |
| 需求确认 | 40% | 明确了客户需求 |
| 方案演示 | 60% | 已展示产品方案 |
| 商务谈判 | 80% | 进入价格/条款协商 |
| 赢单 | 100% | 成功签约 |
| 输单 | 0% | 未能成交 |

**快速记录活动（一句话模式）**：
> "今天给云创王总打了电话，他们用友U8太老了，约下周三演示"

助理会自动解析出：
- 活动类型：电话
- 内容摘要：了解到用友U8版本老旧
- 结果：约定下周三演示
- 建议操作：推进漏斗阶段，更新客户方案信息

**查看漏斗看板**：
> "我的漏斗概览"
> "本周待跟进的客户有哪些"

### 四、销售方法论应用

**生成提问清单**：
> "帮我用SPIN方法论准备云创科技的提问"

**切换方法论**：
> "设置方法论为 BANT"

**三种方法论对比**：

| 方法论 | 核心框架 | 适用场景 |
|--------|----------|----------|
| SPIN | 情境→难点→暗示→效益 | 挖掘深层需求 |
| BANT | 预算→权限→需求→时间 | 快速评估商机质量 |
| MEDDIC | 指标→买家→标准→流程→痛点→支持者 | 复杂大单分析 |

### 五、数据导出与备份

**导出数据**：
> "导出数据"

系统会将 customers、sales_funnel、activity_log 三张表导出为一个 Excel 文件（每表一个 Sheet），提供下载。

**备份提醒**：每周五系统自动提醒您备份数据。

## 注意事项

1. **数据确认**：所有数据修改操作前，助理会先让您确认再执行
2. **隐私保护**：仅使用您提供或合法公开的信息
3. **参谋定位**：所有建议均为参考意见，最终决策权在您手中
4. **数据存储**：数据保存在本地 SQLite 数据库中，请定期备份
5. **搜索增强**：安装 Agent-Reach Skill 后，客户挖掘功能将自动升级为多平台深度搜索，未安装不影响基础功能。
